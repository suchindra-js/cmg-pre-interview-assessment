# CMG Pre-Interview Assessment

Small React + Vite app for the CMG pre-interview assessment.

### How to run the app

Using `pnpm`:

```bash
pnpm install
pnpm dev
```

Using `npm`:

```bash
npm install
npm run dev
```

Visit `https://localhost:5173` in your browser.

### How to run tests

Using `pnpm`:

```bash
pnpm test
```

Using `npm`:

```bash
npm run test
```
